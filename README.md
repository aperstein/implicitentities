# implicitentities

NeurIPS paper materials for the implicit entities project.

## Included files

- `Neurlps/article_v4_neurips.docx`: original Word manuscript
- `Neurlps/article_v4_neurips.tex`: direct Pandoc LaTeX export
- `Neurlps/neurips_2026_template/article_v4_neurips_anonymous.tex`: anonymous NeurIPS-style submission source
- `Neurlps/neurips_2026_template/neurips_2026.tex`: official NeurIPS 2026 template shell
- `Neurlps/neurips_2026_template/neurips_2026.sty`: official NeurIPS 2026 style file
- `Neurlps/neurips_2026_template/checklist.tex`: official checklist file
- `Neurlps/pandoc-media/media/`: extracted figure assets

## Notes

- The anonymous file was adapted from the Word export to match the NeurIPS template structure.
- A LaTeX engine is still needed locally or in Overleaf to compile the project.
